Welcome to RSWiki-Wrapper's documentation!
==========================================

RSWiki-Wrapper is an open-source Python library wrapping functionality of the `RuneScape Wiki <https://runescape.wiki>`_ API. Its goal is to support all RuneScape Wiki API endpoints for both OldSchool RuneScape and RuneScape (aka RS3).

.. note::

   This project is under active development.


To get started using the wrapper, see the :doc:usage section for more information.
